# Changelog

All notable changes to `ntfs-core` (the reader) are documented here. The format
follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.9.2]

- Current published reader: pure-Rust, `forbid(unsafe)`, panic-free-by-lint,
  input-fuzzed NTFS parser — boot sector, MFT records and attributes, runlists,
  index buffers, attribute lists, and LZNT1 decompression.
